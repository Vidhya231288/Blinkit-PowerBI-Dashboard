Blinkit Sales Performance Dashboard
Tools Used: Power BI | DAX | Power Query | Python | Excel
Dataset: Blinkit Grocery Sales Data — 8,523 items across multiple outlet types
Project Overview:
Interactive Power BI dashboard analyzing Blinkit's sales performance across outlet types, locations, item categories, and fat content — enabling data-driven decisions for inventory and sales strategy.
Key KPIs Tracked:

Total Sales: $1.20M
Average Sales per Item: $141
No. of Items: 8,523
Average Rating: 3.9

Dashboard Features:

Filter Panel — Outlet Location, Outlet Size, Item Type slicers
Fat Content analysis (Low Fat vs Regular) — donut chart
Item Type breakdown — horizontal bar chart (15+ categories)
Outlet Establishment trend (2012–2022) — line chart
Outlet Size distribution — donut chart
Outlet Location performance (Tier 1, 2, 3) — bar chart
Outlet Type comparison table (Sales, Items, Avg Sales, Rating, Visibility)

Key Insights:

Tier 3 locations generate highest sales (472.13K)
Fruits & Snack Foods are top-selling item types
Supermarket Type1 dominates with $787.55K total sales
Sales peaked in 2018 at $205K outlet establishment value
Regular fat items slightly outsell Low Fat across all tiers
